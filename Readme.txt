1) simrank.py is the implementation of Simrank with Pruning technique applied.
2) To run
	  ->  python simrank.py 
	  ->  Input two paper's id, to  compare similarity between them 
	  ->  Input -1 -1 to terminate
3) simrank_pruned.py is coded to get top 20 similar papers from the dataset.
4) simrank_top_k.cpp gives the top k similar papers when provided with similarity scores of many papers.
5) Plot is done using python library. (Reference: https://plot.ly/)
